Merge Map

Maps each value to an Observable, then flattens all of these inner Obsevable using mergeAll

Obsevable -> (subscribe =>(res)) --> (subscribe =>(res.data))

Flattening operator -> Obsevable(wit flettening operator) --> (subscribe =>(res.data))

map + mergeAll = mergeMap

mergeAll : Flattens an Obsevable-of-Obsevables

Flattens means : Not subscribe again and again only one time subscribe
